Bank API Test

In this web page you'll see an example of a simple API made with Spring MVC.

Requirements:

  • Client should be able to create an account
  • Client should be able to deposit money in the account
  • Client should be able to withdraw money from the account
  • Client should be able to see account balance and statement

Constraints:

  • For this test the storage of the data should be managed in memory.
  • You should test your code properly with a minimum coverage of 60%
